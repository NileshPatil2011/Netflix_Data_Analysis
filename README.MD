# Netflix Data Analysis

A short exploratory analysis of Netflix movie ratings using a Jupyter Notebook.

## Contents
- `src/Netflix_Data_Analysis.ipynb` — Notebook performing data loading, basic statistics, and visualizations.
- `data/Netflix_Movie_Data.xls` — Source dataset (expected path used by the notebook).

## Quick start
1. Install dependencies: `pandas`, `numpy`, `matplotlib`, `seaborn`, and `jupyter`.
2. From the repository root, launch the notebook:
   - jupyter notebook src/Netflix_Data_Analysis.ipynb
3. Run the cells to reproduce the analysis and plots.

## Notes
- The notebook expects the dataset at `data/Netflix_Movie_Data.xls`. Adjust the path in the notebook if your file is located elsewhere.
